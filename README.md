# Image Enhancer

Система улучшения изображений в реальном времени с использованием машинного обучения прямо в браузере.

## Возможности

- **AI-анализ**: Нейросеть автоматически подбирает оптимальные параметры коррекции
- **Быстрая обработка**: Среднее время обработки ~5 секунд
- **Приватность**: Все вычисления происходят локально в браузере
- **Кроссплатформенность**: Работает во всех современных браузерах
- **Три параметра**: Яркость, контрастность и цветность

## Быстрый старт

### Требования

- **Node.js** ([скачать](https://nodejs.org/))
- Современный браузер с поддержкой WebGL2

### Установка и запуск

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/MIFomin//image-enhancer.git
cd image-enhancer

# 2. Установите зависимости (WASM файлы скопируются автоматически)
npm install

# 3. Запустите проект
npm run dev
```

### Обучение собственной модели
Если хотите переобучить модель на своих данных:

```bash
# 1. Установите Python зависимости
pip install tensorflow numpy matplotlib pillow scipy tf2onnx onnx

# 2. Скачайте датасет
cd python
python 1_download_dataset.py

# 3. Обучите модель
python 2_train_model.py

# 4. Экспортируйте в ONNX
python 3_export_model.py
```
